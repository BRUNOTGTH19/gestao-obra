# Gestao Obra

Sistema de gestao de obras com controle de funcionarios, gastos e orcamentos.

## Tecnologias

- **Backend:** Fastify, TypeScript, Prisma, MySQL
- **Frontend:** React, Vite, TypeScript
- **Deploy:** Render (API) + Vercel (Frontend)

## Variaveis de Ambiente

### Backend
Copie \backend/.env.example\ para \backend/.env\ e preencha com seus valores.

### Frontend
Copie \frontend/.env.example\ para \frontend/.env\ e preencha com seus valores.

## Rodando localmente

### Backend
\\\bash
cd backend
npm install
npm run dev
\\\`n
### Frontend
\\\bash
cd frontend
npm install
npm run dev
\\\`n
